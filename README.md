# Numerical Suffix

This package is used to add metric suffixes to numbers.

## Installation

```
npm install @auth0/numerical-suffix
```

## Usage

### ES2015+

```
import numericalSuffix from '@auth/numerical-suffix'
```

### NodeJS / CommonJS

```
const numericalSuffix = require('auth0/numerical-suffix')
```
